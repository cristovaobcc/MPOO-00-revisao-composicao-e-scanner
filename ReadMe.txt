A ideia desse programa é abordar 2 pontos:

1) Composição de objetos; e

2) Problemas com uso da classe Scanner.

Quanto ao item (1), deve-se atentar ao seguinte:
Nenhum livro (todo) é criado(cadastrado) sem ter uma estante(parte) a qual pertença.
Daí que, na classe Estante, eu forcei o uso do construtor do todo(livro) no 
construtor da parte(estante).

Quanto ao item (2), deve-se atentar aos commits onde discuto os alguns problemas 
enfretados pelo uso na classe Scanner para ler do teclado.
